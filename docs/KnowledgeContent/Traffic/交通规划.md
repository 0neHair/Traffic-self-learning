# 交通规划

## 课程简介

???+ note "2019级 课程简介"

    交通规划课程主要包含两个环节： 
    
    1. 课堂授课；
    2. 基于TransCAD和Transtar的实验操作；

## 先修要求

数学基础课、交通系统分析

## 开设学期

大三上

## 考核方式

闭卷考试
*[闭卷考试]: 计算题都不难，剩下的就是背书，严格的背书。
