# 交通行为分析

## 课程简介

???+ note "2019级 课程简介"

    交通行为分析包含三部分内容: 

    1. 离散选择模型；
    2. SP 调查；
    3. 博弈论

## 先修要求

数学基础课

## 开设学期

大二下

## 考核方式
