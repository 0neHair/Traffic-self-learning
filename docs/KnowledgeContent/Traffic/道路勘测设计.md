# 道路勘测设计

## 课程简介

???+ note "2019 级课程简介"

    本课程包含以下内容包含绪论、平面设计、纵断面设计、横断面设计、总体设计、选线与定线、道路交叉设计等部分

## 先修要求

- 交通运输导论
- 土木工程测量
- 画法几何及 CAD 制图

## 开设学期

大二下
