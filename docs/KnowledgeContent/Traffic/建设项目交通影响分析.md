# 建设项目交通影响分析

## 课程简介

???+ note "2019 级 课程简介"

    偏介绍性的课程，研究造房子（住宅、商场等）如何影响附近的交通、如何改进（出入口怎么设计、附近街道怎么设计等）

## 开设学期

大三下

## 考核方式

小组大作业，基于项目基础资料信息，结合框架目录，形成完整规范的交评报告，包括两部分：

- 汇报
*[汇报]: 制作PPT汇报交评报告

- 报告
*[报告]: 一篇完整的交评报告
