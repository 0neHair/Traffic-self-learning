# Traffic-Safety

## 课程简介

???+ note "2019级 课程简介"

    这是一门全英文授课课程，共包含三部分内容:

    1. 统计分析；
    2. 逻辑回归；
    3. Before-after Study

每一部分都有对应的 Case Study

## 先修要求

- 数学类基础课
- 交通大数据分析

## 开设学期

大三上

开卷考试
