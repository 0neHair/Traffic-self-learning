# 驾驶行为与心理

## 课程简介

???+ note "2019 级 课程简介"

    课程时长8周，2019年为该课程开设的第一年，课程内容较为新，课程无教材，以PPT和近几年的文献为主，主要课程内容包括
    - 识别、预测危险驾驶行为
    - 驾驶行为和心理对交通安全的影响
    - 辅助驾驶与自动驾驶

## 开设学期 

大三下

## 考核方式 

汇报
*[汇报]: 3-5人为小组，在老师给定的选题范围内，选择感兴趣的方向进行文献研究，制作PPT进行展示汇报
