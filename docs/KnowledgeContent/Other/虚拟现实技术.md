# 虚拟现实技术

## 课程简介

???+ note "2019 级 课程简介"

    课程时长8周，课程教材为《虚拟现实技术》，主要课程内容包括：

    - 虚拟现实技术介绍
    - 一点点3D建模和虚拟现实技术编程

## 先修要求

一定的编程基础

## 开设学期

大三下

## 考核方式

课程论文
*[课程论文]: 以虚拟现实技术在交通运输工程领域或相关领域的应用为选题背景，自拟题目，撰写一篇科技论文

小组大作业
*[小组大作业]: 小组完成一个虚拟现实的场景，采用Unity或虚幻引擎都行
